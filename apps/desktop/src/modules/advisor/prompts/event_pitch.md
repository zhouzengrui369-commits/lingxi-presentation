# 活动宣讲 Prompt

## 场景

路演 / 发布会 / 分享会 / 校园宣讲 — 各种对外/对内的宣讲场景。

## 顾问提问清单（3 题）

1. **类型**：投资人路演 / 产品发布 / 技术分享 / 校园招聘
2. **风格**：专业严谨 / 激情感染 / 故事化叙事
3. **期望的现场行动**（文本题，可空）：扫码注册 / 立即试用 / 留联系方式

## 知识库关联信号

- type=投资人路演 → 推荐"专业严谨"风格
- type=产品发布 → 推荐"激情感染"风格
- type=技术分享 → 推荐"故事化叙事"风格
