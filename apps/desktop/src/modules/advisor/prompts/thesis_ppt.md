# 答辩 PPT Prompt

## 场景

毕业论文 / 课程答辩，时长通常 10-30 分钟，委员会关注"研究问题 → 方法 → 创新 → 表达"。

## 顾问提问清单（3 题）

1. **类型**：本科 / 硕士 / 博士 / 课程
2. **时长**：10 / 20 / 30 分钟
3. **重点章节**（多选）：背景 / 方法 / 结果 / 创新点

## 知识库关联信号

- type=博士 → 自动加上"未来工作 / 研究展望"章节
- duration=10min → 限制背景不超过 1 页
- focus=创新点 → 在结尾加 "Future Work" 1-2 页

## 答辩黄金结构（前 30 秒决定第一印象）

```
封面 (1) → 背景与动机 (1-2) → 相关工作 (1) → 方法/实现 (3-5)
→ 实验结果 (3-5) → 创新点 (1-2) → 局限与展望 (1) → 致谢 (1)
```
