# 季度汇报 PPT Prompt

## 场景

季度汇报是 PRD 定义的 First-use 场景，受众最广（部门同事 / 管理层 / 全员 / 外部合作伙伴），数据密度最高。

## 顾问提问清单（4 题，覆盖主题/受众/页数/数据深度）

1. **主题**：业务增长 / 产品迭代 / 团队建设 / 运营效率
2. **受众**：部门同事（kb_default）/ 管理层 / 全员 / 外部
3. **页数**：精简 (8-12) / 标准 (15-20) / 详尽 (25-35)
4. **数据深度**（多选）：关键指标趋势 / 同比环比 / OKR / 用户分群

## 知识库关联信号

- 主题选"业务增长" → 受众默认推荐"管理层"（向上汇报场景）
- 主题选"团队建设" → 受众默认推荐"全员大会"
- 主题选"运营效率" → 数据深度推荐"关键指标趋势"
- 受众选"部门同事" → 弱化财务细节，强调项目进展

## 输出大纲建议（基于回答生成）

```
第 1 页: 封面（主题 + 季度 + 汇报人）
第 2-3 页: 核心结论摘要（3-5 个 bullet）
第 4-7 页: 关键指标趋势（图表）
第 8-10 页: 主要项目进展
第 11-13 页: 问题与反思
第 14-15 页: 下季度 OKR 展望
第 16 页: 致谢 / Q&A
```
