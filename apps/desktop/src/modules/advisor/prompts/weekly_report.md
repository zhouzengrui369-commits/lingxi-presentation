# 周报 Prompt

## 场景

周报覆盖个人/项目组/部门三种 scope，重点是"快速让上级知道这周发生了什么"。

## 顾问提问清单（3 题）

1. **范围**：个人 / 项目组 / 部门
2. **本周重点**：交付 / 事故排查 / 会议协调 / 调研
3. **输出格式**（多选）：Markdown / HTML / 飞书文档

## 知识库关联信号

- scope=部门 → 默认推荐 "Markdown 简版"（便于快速传阅）
- focus=事故 → 触发额外追问"事故严重等级"
- focus=调研 → 推荐 "飞书文档"（便于多人协作批注）

## 周报四段式模板

```
## 本周完成
- ...

## 进行中
- ...

## 阻塞
- ...

## 下周计划
- ...
```
