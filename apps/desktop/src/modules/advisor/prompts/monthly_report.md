# 月度运营报告 Prompt

## 场景

月度运营报告是 ToB 高频场景，受众为 CEO/业务负责人/投资方，核心是数据驱动决策。

## 顾问提问清单（3 题）

1. **交付对象**：CEO/管理层 / 业务负责人 / 投资方
2. **核心指标**（多选）：营收 / 用户增长 / 留存 / 成本
3. **是否需要同比/环比**

## 知识库关联信号

- audience=投资方 → 默认推荐"同比/环比"（外部汇报必备）
- audience=CEO → 默认推荐"营收 + 用户增长"
- 选择"营收" → 联动推荐"成本毛利"对比

## 月报结构建议

```
1. 本月核心摘要 (1 页)
2. KPI 趋势图 (营收 / 用户 / 留存)
3. 同比 / 环比
4. 重点事件 / 里程碑
5. 风险与挑战
6. 下月计划
```
